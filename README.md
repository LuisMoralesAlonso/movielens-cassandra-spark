# movielens-cassandra-spark
Pruebas de Clusters de Cassandra y Spark en base al set de datos de movielens.

Cluster Instalado.
Apache Cassandra 2.1.9
Apache Spark 1.3.1.
OpsCenter 5.2.0
Tamaño del cluster: 3 nodos.
En un nodo adicional he instalado OpsCenter y el master de Spark.

Características de las máquinas (pc's de sobremesa):
4 cores.
16 GB RAM.
200 GB HDD.

Software Base:
CentOS 7
